# 项目简介

基于Springboot+vue人事管理平台是一款集成了现代技术栈的人事管理系统。该平台旨在简化人事管理流程，提高管理效率。

## 核心功能
- 管理员登录模块：超级管理员拥有权限进行系统管理，包括部门、培训、招聘、请假、员工、考勤和薪酬管理。
- 部门管理模块：支持部门管理列表的查看，增加、删除、修改和查询操作。
- 用户管理模块：支持用户列表的数据查看，包括增加、删除、修改和查询操作。
- 培训管理列表管理模块：支持培训管理列表的数据查看，并对培训进行修改操作。

## 适用场景

- 适用于企业、事业单位的人事管理部门，用于简化人事管理流程。
- 可应用于部门、培训、招聘、请假、员工、考勤和薪酬等管理需求。
- 支持多用户操作，提高管理效率。

## 技术特点

- 采用Springboot、vue、mybatisplus、mysql等技术栈，保证系统的稳定性和高效性。
- 可使用idea、Navicat、VScode等工具进行开发调试。
- 附赠6000字文档，便于用户快速上手和了解系统。
